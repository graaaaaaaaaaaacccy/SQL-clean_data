Project Overview

This project focuses on cleaning and analyzing a messy e-commerce dataset using SQL. The goal was to transform raw, unstructured data into a clean and reliable dataset, then extract meaningful insights that can support business decision-making.

 Objectives
Clean and prepare raw data for analysis
Handle missing values and inconsistencies
Remove duplicates and ensure data accuracy
Perform exploratory data analysis (EDA)
Generate insights on sales performance and customer behavior
Data Cleaning Process
The dataset initially contained missing values, duplicates, and inconsistent formatting. The following steps were taken:

Created separate tables for raw and cleaned data
Removed duplicate records using SQL
Handled missing values:
Removed rows where product_name was NULL (critical field)
Replaced NULL values in customer_city and payment_method with 'Unknown'
Standardized text fields (e.g., payment methods like M-Pesa)
Verified data quality after cleaning

- SQL (data cleaning, aggregation, analysis)
